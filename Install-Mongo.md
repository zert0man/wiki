# Установка сервера Mongo DB

Если у вас по каким-то причинам не работает виртуальная машина, вы можете установить MongoDB на свой компьютер.

1.  Зайдите на официальную страницу скачивания и скачайте бинарные файлы в соответствии с вашей операционной системой.
1.  Распакуйте архив в любое удобное место на диске и перейдите в папку `bin.  `mongod` - это сервер, а `mongo` - клиентская консоль.
1.  Для удобства запуска создайте скрипт (bat-файл), который будет передавать серверу параметр - место, где лежат файлы базы. Например, для Windows:

	```
	bin\mongod.exe --dbpath=c:\projects\mongodb\db
	```
	
    Убедитесь, что указанный вами путь `dbpath` существует.
1.  Запустите созданный скрипт. Появится черное окно сервера.
1.  Запустите консоль. Теперь вы можете выполнять лабораторную работу
